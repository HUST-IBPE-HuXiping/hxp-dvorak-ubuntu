# xmodmap config used by Hu Xiping

## Changed the keyboard layout to a more favorable one
* TAB -> Page Down
* CapsLock -> TAB
* LCtrl -> LSuper
* LSuper -> LCtrl
* Menu -> RCtrl
* RCtrl -> Page Up

## How to Deploy
* On Ubuntu, copy this file to a location, use command xmodmap .xmodmap to start
* Better add this command to startup
